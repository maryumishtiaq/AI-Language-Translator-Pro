# 🌍 AI Language Translator Pro

A modern AI-powered Language Translator built using **Python** and **Streamlit**.

## 🚀 Features

- 🌍 Translate text into 100+ languages
- 🔄 Auto Detect Language
- 🌗 Dark & Light Theme
- 📋 Copy Translation
- 📄 Download Translation as TXT
- 📑 Export Translation as PDF
- 🔊 Text-to-Speech
- ⭐ Favorite Languages
- 📜 Translation History
- 📊 Translation Statistics
- 🎨 Modern Glassmorphism UI

---

## 🛠 Technologies Used

- Python
- Streamlit
- Deep Translator
- gTTS
- FPDF2
- Pyperclip

---

## 📦 Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/AI-Language-Translator-Pro.git
```

Go to project folder

```bash
cd AI-Language-Translator-Pro
```

Install requirements

```bash
pip install -r requirements.txt
```

Run the project

```bash
streamlit run translator.py
```
